# Documentação OpenRouter\n\nObtida de: https://openrouter.ai/docs\n\n## Guia Rápido OpenRouter\n\nOpenRouter fornece uma API unificada que dá acesso a centenas de modelos de IA através de um único endpoint, lidando automaticamente com fallbacks e selecionando as opções mais econômicas.\n\n### SDK OpenRouter (Beta)\n\nInstalação:\n```bash\nnpm install @openrouter/sdk\n```\n\nExemplo TypeScript:\n```typescript\nimport { OpenRouter } from '@openrouter/sdk';\n\nconst openRouter = new OpenRouter({\n  apiKey: '<OPENROUTER_API_KEY>',\n  defaultHeaders: {\n    'HTTP-Referer': '<YOUR_SITE_URL>',\n    'X-Title': '<YOUR_SITE_NAME>',\n  },\n});\n\nconst completion = await openRouter.chat.send({\n  model: 'openai/gpt-4o',\n  messages: [{ role: 'user', content: 'Qual é o sentido da vida?' }],\n  stream: false,\n});\n\nconsole.log(completion.choices[0].message.content);\n```\n\n### API Direta (Python)\n\n```python\nimport requests\nimport json\n\nresponse = requests.post(\n  url=\"https://openrouter.ai/api/v1/chat/completions\",\n  headers={\n    \"Authorization\": \"Bearer <OPENROUTER_API_KEY>\",\n    \"HTTP-Referer\": \"<YOUR_SITE_URL>\",\n    \"X-Title\": \"<YOUR_SITE_NAME>\",\n  },\n  data=json.dumps({\n    \"model\": \"openai/gpt-4o\",\n    \"messages\": [{\"role\": \"user\", \"content\": \"Qual é o sentido da vida?\"}]\n  })\n)\n```\n\n### SDK OpenAI\n\nExemplo TypeScript:\n```typescript\nimport OpenAI from 'openai';\n\nconst openai = new OpenAI({\n  baseURL: 'https://openrouter.ai/api/v1',\n  apiKey: '<OPENROUTER_API_KEY>',\n});\n\nconst completion = await openai.chat.completions.create({\n  model: 'openai/gpt-4o',\n  messages: [{ role: 'user', content: 'Qual é o sentido da vida?' }],\n});\n```\n\nSuporta streaming e integrações com outros SDKs.
