**[Database](https://supabase.com/docs/guides/database/overview)**: Supabase provides a full Postgres database for every project with Realtime functionality, database backups, extensions, and more.\n- **[Auth](https://supabase.com/docs/guides/auth)**: Add and manage email and password, passwordless, OAuth, and mobile logins to your project through a suite of identity providers and APIs.\n- **[Storage](https://supabase.com/docs/guides/storage)**: Store, organize, transform, and serve large files—fully integrated with your Postgres database with Row Level Security access policies.\n- **[Realtime](https://supabase.com/docs/guides/realtime)**: Listen to database changes, store and sync user states across clients, broadcast data to clients subscribed to a channel, and more.\n- **[Edge Functions](https://supabase.com/docs/guides/functions)**: Globally distributed, server-side functions to execute your code closest to your users for the lowest latency.\n\n### Postgres
